# Tipos de Instruções

## Expressão literal, retorna o próprio número
```python
>>> 1 
1
```
## Expressão, retorna o proprio texto
```python
>>> "Alexandre"
'Alexandre
```
## Espressão com operador, retorna um resultado booleano
```python
>>> 1 == 1
True
```
```python
>>> 89 >= 89
True
```
## Chamada de função
```python
>>> print("Alexandre")
Alexandre
>>> sum([1, 2, 3])
6
``` 
# Declarações (Statements)
## Condicional com comparação
```python
if 1 > 2;
    print("Um é maior do que dois")
```
## Comparação com comparação de igualdade
```python
if x == y;
    # faça isso
else
    faça aquilo
```
## Repetição com comparação de valor
```python
while numero < 10
    numero += 1
    print(numero)
```
## Instrução nula
```python
if nao_faca_nada;
    pass
```

# Atribuição (ASssignment)
```python
preco = 10
quantidade = 5
total = preco * quantidade
print(f"O total da sua compra é {total}")
```
